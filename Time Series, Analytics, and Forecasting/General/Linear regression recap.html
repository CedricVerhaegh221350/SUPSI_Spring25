Derivation of the closed-form solution

Let's denote XXX﻿ as the matrix of input features, where each row represents a data point and each column represents a feature. The vector yyy﻿ represents the observed outputs for each data point. Our goal is to find the parameter vector β\betaβ﻿ (containing the weights of the regression model) that minimizes the sum of squared errors between the predicted outputs XβX\betaXβ﻿ and the actual outputs y yy﻿​
The sum of squared errors can be expressed as a function of β\betaβ﻿:
J(β)=(y−Xβ)⊤(y−Xβ)J(\beta) = (y-X\beta)^\top(y-X\beta)J(β)=(y−Xβ)⊤(y−Xβ)
To find the minimum of J(β)J(\beta)J(β)﻿, we take its derivative with respect to β\betaβ﻿ and set it to zero:
∂J(β)∂β=−2X⊤(y−Xβ)=0 \frac{\partial J(\beta)}{\partial \beta} = -2X^\top (y - X\beta) = 0 ∂β∂J(β)​=−2X⊤(y−Xβ)=0
Solving this equation for β\betaβ﻿ gives us the closed-form solution:
β=(X⊤X)−1X⊤y\beta = (X^\top X)^{-1}X^\top yβ=(X⊤X)−1X⊤y
This solution is known as the Normal Equation in linear regression. It provides a direct way to compute the optimal weights β\betaβ﻿ that minimizes the sum of squared errors between the predicted and actual outputs. The term (X⊤X)−1X⊤(X^\top X)^{-1}X^\top(X⊤X)−1X⊤﻿ is often referred to as the Moore-Penrose pseudoinverse. This closed-form solution is elegant and computationally efficient for problems with a small to moderate number of features but can become computationally expensive or numerically unstable for very large datasets or when X⊤XX^\top XX⊤X﻿ is nearly singular.

Interpretation of the closed-form solution

The Moore-Penrose pseudoinverse allows us to find a one-shot solution to the linear regression problem since linearly de-correlates columns of X XX﻿. To see this, we can directly find an orthonormal transformation CCC﻿ which generates a diagonal covariance for XXX﻿:
C⊤X⊤XC=ΛC^\top X^\top XC = \LambdaC⊤X⊤XC=Λ
Since CCC﻿ is orthonormal, we can write:
X⊤X=CΛC⊤X^\top X = C\Lambda C^\topX⊤X=CΛC⊤
which is the diagonalization of the Sum of Squares and Cross Products (SSCP) matrix. Since diagonalization is unique, CCC﻿'s columns are eigenvectors of X⊤XX^\top XX⊤X﻿ and Λ\LambdaΛ﻿'s diagonal elements are its eigenvalues.
We can rewrite the linear regression solution as:
β=(X⊤X)−1X⊤y=CΛ−1C⊤X⊤y=CΛ−1(XC)⊤y=CΛ−1β∗\begin{aligned} \beta &= (X^\top X)^{-1}X^\top y \\ &= C \Lambda^{-1}C^\top X^\top y\\ &= C\Lambda^{-1}\color{green}{(XC)^\top}\color{balck}{y} \\ &= C\Lambda^{-1}\color{violet}{\beta^*} \end{aligned}β​=(X⊤X)−1X⊤y=CΛ−1C⊤X⊤y=CΛ−1(XC)⊤y=CΛ−1β∗​
which can be interpreted in this way:
we first transform XX X﻿ by pre-multiplying it with CCC﻿ de-correlating its columns.
Then we find the linear regression coefficient for the decorrelated matrix XCXCXC﻿, β∗\beta^*β∗﻿​
This can be achieved just by performing the scalar product with yyy﻿ since columns of XCXCXC﻿ are uncorrelated
At last, we rescale and re-correlate the linear coefficients β∗\beta^*β∗﻿ through CΛ−1β∗C\Lambda^{-1}\beta^*CΛ−1β∗﻿.
Inspecting the formula above, we can infer that decorrelating a matrix’s columns is equivalent to find the inverse of its SSCP matrix.
